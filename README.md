# Journal de Sport App 🏋️

Application Android native permettant de suivre ses entraînements sportifs au quotidien : séances, exercices, séries, répétitions (reps) et charges.

## 📱 Fonctionnalités

- Enregistrement des séances d'entraînement
- Ajout d'exercices avec séries, répétitions (reps) et poids
- Historique des performances pour suivre sa progression
- Interface simple et intuitive

## 🛠️ Stack technique

- **Langage** : Java
- **IDE** : Android Studio
- **Build system** : Gradle (Kotlin DSL)

## 🚀 Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/maty382/maty382-journal-de-sport-app.git
   ```
2. Ouvrir le projet dans Android Studio
3. Laisser Gradle synchroniser les dépendances
4. Lancer l'application sur un émulateur ou un appareil physique

## 📖 Glossaire

- **Reps** : abréviation de "répétitions", le nombre de fois qu'un mouvement est effectué dans une série
- **Sets** (Séries) : le nombre de fois qu'un groupe de mouvements est répété

## 📂 Structure du projet

```
SportJournal/

├── gradle/             # Wrapper Gradle
```
